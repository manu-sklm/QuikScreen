  Admin
</NavLink>

Now, the link is only active if the path is exactly /admin â€” not for /admin/....

----------------------------------
If you donâ€™t use the end prop, then:

  Admin
</NavLink>


This link will be marked as active not only for /admin, but also for any route that starts with /admin, like:

/admin/dashboard

/admin/add-shows

/admin/list-bookings
